This is a sample training document about Machine Learning.

Machine Learning is a subset of artificial intelligence (AI) that focuses on building systems that learn from data.

Key concepts:
1. Supervised Learning - Learning with labeled data
2. Unsupervised Learning - Finding patterns in unlabeled data
3. Neural Networks - Computing systems inspired by biological neural networks
4. Deep Learning - Uses multiple layers to extract features from raw input

Applications include image recognition, natural language processing, and predictive analytics.
